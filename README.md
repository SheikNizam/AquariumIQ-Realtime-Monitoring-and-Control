🐠 AquariumIQ – Smart Aquarium Monitoring and Control System

A smart IoT-based aquarium management system designed to automate aquarium maintenance by continuously monitoring water conditions and controlling essential aquarium equipment. The system uses an ESP32 microcontroller to collect sensor data, automate operations, and provide real-time monitoring through a mobile application.

📖 Overview

AquariumIQ is developed to reduce manual effort involved in aquarium maintenance while ensuring a healthy environment for aquatic life. The system continuously monitors important water parameters and automatically controls feeding, lighting, water circulation, and temperature. Sensor data is transmitted over Wi-Fi and stored in a cloud database for remote monitoring through a mobile application.

✨ Features
🌡️ Real-time water temperature monitoring
🧪 pH monitoring for water quality
🐟 Automatic fish feeding
💧 Automatic water circulation control
💡 Smart lighting control
🌡️ Automatic temperature control
📱 Mobile application for remote monitoring
☁️ Cloud database integration
📊 Live sensor data visualization
📶 Wi-Fi based IoT connectivity
